## Extract a single file from a tar archive

The archive **{{ARCHIVE}}** contains a full project tree with several files
and subdirectories.

Your task:

1. Extract **only** the file **{{TARGET_FILE}}** from **{{ARCHIVE}}**.
2. Place it under **{{EXTRACT_DIR}}**, preserving its directory structure
   (e.g. the result should be `{{EXTRACT_DIR}}/{{TARGET_FILE}}`).
3. Do **not** extract the rest of the archive's contents.
4. Leave **{{ARCHIVE}}** itself untouched.
