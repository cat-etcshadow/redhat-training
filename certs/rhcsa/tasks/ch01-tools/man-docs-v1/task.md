## Locate and use system documentation

The exam environment has no internet access. You must use `man` pages and
on-system documentation to find the options you need.

Your task:

1. Copy `/etc/hosts` to **{{DEST_FILE}}**, preserving permissions, timestamps,
   ownership, and symbolic links.

2. Find all files in `/var/log` larger than **1 MB** and write the list to
   **{{OUTPUT_FILE}}**.
