## Create a cron job for a specific user

The user **{{CRON_USER}}** (already exists) must run a script automatically.

Your task:

1. Create a cron job for user **{{CRON_USER}}** that runs
   `{{CRON_SCRIPT}}` every day at **{{CRON_HOUR}}:{{CRON_MIN}}**.

2. The cron job must run as the user **{{CRON_USER}}**, not as root.

3. The script `{{CRON_SCRIPT}}` already exists and is executable.
