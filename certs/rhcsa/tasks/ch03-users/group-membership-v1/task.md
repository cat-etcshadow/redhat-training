## Manage group membership

Your task:

1. Create user **{{TARGET_USER}}** if it does not already exist.
2. Create supplementary groups **{{SUPP_GROUP1}}** and **{{SUPP_GROUP2}}** if they do not exist.
3. Create group **{{NEW_PRIMARY}}** if it does not exist, then change
   **{{TARGET_USER}}'s primary group** to **{{NEW_PRIMARY}}**.
4. Add **{{TARGET_USER}}** as a supplementary member of **{{SUPP_GROUP1}}** and **{{SUPP_GROUP2}}**
   (do not overwrite existing supplementary groups).
