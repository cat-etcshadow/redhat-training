## Batch-manage group membership with gpasswd

The group **{{TARGET_GROUP}}** exists and currently has one member:
**{{OLD_MEMBER}}**. The users **{{NEW_MEMBER1}}**, **{{NEW_MEMBER2}}**, and
**{{NEW_MEMBER3}}** already exist but are not members of the group.

Your task:

Using a **single command**, set the membership of **{{TARGET_GROUP}}** to
be exactly **{{NEW_MEMBER1}}**, **{{NEW_MEMBER2}}**, and **{{NEW_MEMBER3}}**
— replacing the existing membership list entirely. **{{OLD_MEMBER}}** must
no longer be a member afterward.
